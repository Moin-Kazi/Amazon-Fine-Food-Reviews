# Amazon Fine Food Reviews Analysis

## Objective
Analyze Amazon Fine Food reviews to understand customer sentiment and product feedback.

## Dataset
Amazon Fine Food Reviews dataset (Kaggle).
Used sample data for analysis.

## Tools & Skills
- Python
- Pandas, NumPy
- NLTK
- Machine Learning (basic)
- Jupyter Notebook

## Key Work
- Text cleaning & preprocessing
- Sentiment analysis
- Data visualization

## Result
Extracted positive, negative, and neutral customer sentiments to understand product quality.
